# wireshark-packet-analysis-report
Performed network packet analysis using Wireshark to capture and examine ARP, DNS, TCP, TLS, and QUIC traffic. Analyzed packet structures, identified network anomalies, and documented findings with security assessments and recommendations.


## Tools Used
- Wireshark 4.6.4
- Npcap 1.86
- Windows 11

## Key Features
- DNS Traffic Analysis
- ARP Packet Analysis
- TCP Communication Analysis
- TLS/QUIC Traffic Inspection
- Suspicious Packet Detection
- Security Assessment Report

## Outcome
The analysis identified normal network communication patterns with no evidence of malicious activity, ARP poisoning, DNS tunneling, or unauthorized access.
