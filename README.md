# Wireshark Packet Analysis

This project demonstrates network traffic analysis using Wireshark. The packet capture includes DNS, ARP, TCP, TLS, and QUIC traffic. The objective was to identify protocols, analyze packet structures, investigate suspicious traffic, and document security findings.

## Tools Used
- Wireshark 4.6.4
- Npcap 1.86
- Windows 11

## Key Features
- DNS Traffic Analysis
- ARP Packet Analysis
- TCP Communication Analysis
- TLS/QUIC Traffic Inspection
- Suspicious Packet Detection
- Security Assessment Report

## Outcome
The analysis identified normal network communication patterns with no evidence of malicious activity, ARP poisoning, DNS tunneling, or unauthorized access.
